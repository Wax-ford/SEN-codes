Project Overview

The Student Result Management System (SRMS) is a Python-based application designed to help lecturers and school administrators record student details, compute academic results, and assign grades based on performance.

Features

- Student Information Collection: Capture student name and matriculation number
- Grade Calculation: Automatically compute total scores and assign grades
- Result Display: Present student results in a clear, formatted output
- Simple Interface: User-friendly command-line interaction

Grading System

The system uses the following grading scale based on total score:
- A: 70 marks and above
- B: 60 - 69 marks
- C: 50 - 59 marks
- D: 45 - 49 marks
- F: Below 45 marks
